# Automatic-Multi-Sensor-Liquid-Tank-Control-System-Using-Arduino-Uno
Repository ini berisi perancangan sistem pengendalian cairan otomatis berbasis Arduino Uno dengan multi-sensor (suhu, level, dan tekanan). Sistem mengontrol pompa, katup, dan alarm secara close loop, menampilkan data pada LCD, serta dilengkapi diagram blok dan P&amp;ID sesuai standar instrumentasi.
